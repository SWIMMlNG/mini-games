## 🚀 本地运行

### 方式一：Python 简易服务器
```bash
cd mini-games
python3 -m http.server 8080
# 浏览器访问 http://localhost:8080
```

### 方式二：VS Code Live Server
1. 在 VS Code 中打开项目
2. 安装 Live Server 插件
3. 点击 "Go Live" 按钮

---

## 🛠️ 技术栈

- HTML5
- CSS3 (渐变、动画、响应式布局)
- JavaScript (Canvas API)
- Python (Pygame - 桌面版)

## 📁 项目结构

```
mini-games/
├── index.html              # 游戏大厅首页
├── README.md               # 项目说明
└── snake_game/             # 贪吃蛇游戏
    ├── index.html          # 网页版
    ├── snake_game.py       # Python桌面版
    └── run_snake.sh        # 启动脚本
```

---

## 🎮 添加新游戏

1. 在根目录创建新文件夹（如 `tetris/`）
2. 添加 `index.html` 作为游戏入口
3. 在根目录 `index.html` 中添加游戏卡片
4. 提交并推送到 GitHub
